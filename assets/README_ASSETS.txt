ASSETS FOLDER
==============

This folder contains application assets such as icons.

ICON CUSTOMIZATION:
-------------------
To add a custom icon to your application:

1. Create or download a 256x256 pixel icon image
2. Convert it to .ico format using an online tool:
   - https://convertio.co/png-ico/
   - https://icoconvert.com/
   - Or use GIMP/Photoshop
3. Save it as "icon.ico" in this folder
4. Rebuild the executable using build_exe.bat

CURRENT ICON:
-------------
A simple placeholder icon is generated automatically.
You can replace it with your own professional icon for submission.

RECOMMENDED ICON SPECS:
-----------------------
- Format: .ico (Windows Icon)
- Size: 256x256 pixels
- Multiple resolutions embedded: 16x16, 32x32, 48x48, 256x256
- Simple, recognizable design
- Related to file management/folders

ICON IDEAS:
-----------
- Folder with arrows (representing renaming)
- Files with editing symbols
- Document stack with pencil
- Abstract "transform" symbol
